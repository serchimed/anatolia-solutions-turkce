Kitap Hakkında

Yapay Zeka ile Konuşmak Rehberi, yapay zekayla etkili iletişim kurmak isteyenler için sürekli güncellenen bir kaynaktır. Temel kavramlardan etik konulara, yaratıcı uygulamalardan gerçek dünya örneklerine kadar geniş bir yelpazeyi sade bir dille açıklar.

Komut (prompt) tasarımı becerilerinizi geliştirmek için pratik çerçeveler, örnekler ve alıştırmalar sunar. Hem yeni başlayanlar hem de deneyimli kullanıcılar için uygundur.

Rehberin ana odak noktalarından biri prompt mühendisliğidir. Okuyucular, şu teknikleri keşfedeceklerdir:

  • Sıfır Vuruşlu Prompt (Zero-Shot Prompting)
  • Az Örnekli Prompt (Few-Shot Prompting)
  • Zincirleme Yapıda Prompt (Interactive Prompt Chaining)
  • Düşünce Zincirli Prompt (Chain of Thought Prompting)
  • Düşünce Ağacı Yapısında Prompt (Tree of Thoughts Prompting)
  • Promptu Yeniden Şekillendirme (Prompt Reframing)
  • Kendinden Tutarlı Yönlendirmeli Prompt (Self Consistency Prompting)
  • Doğrudan Uyarıcı Prompt (Direct Stimulus Prompting)
  • Talimat Tabanlı Prompt (Instruction Based Prompting)
  • Benzetme Tabanlı Prompt (Analogical Reasoning / Association Prompting)
  • Prompt Oluşturan Prompt (Meta Prompting)
  • Şablon Tabanlı Prompt (Template Based Prompting)
  • Karşılaştırmalı Prompt (Contrastive / Comparison / Contrast Prompting)
  • Keşifsel Prompt (Exploratory Prompting)
  • Koşullu Prompt (Conditional Prompting)
  • Geniş Alan Aktifleştiren Prompt (Latent Space Activation Prompting)
  • Rol Yapma Yapısında Prompt (Role Playing / Scenario-Based Prompting)

Çalışma rehberi, okuyuculara en güncel stratejileri sunarak sürekli olarak yeni teknikler, bilgiler ve örneklerle genişlemektedir. Yeni bölümler talep üzerine okurlara sunulur.
